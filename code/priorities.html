--- layout: post title: Hover Effect Feature Row ---

This feature row highlights the three main service priorities of Texas A&M Health. The text, images, and graphics were all previously used in print marketing materials, and I wanted to build a component that combined all of the elements in an interactive way.

I wrote the HTML for small screens first, originally using the images rather than graphics. Once the graphics were finalized for web, I combined the two to create the medium screen layout, and finally incorporated a Bootstrap hover effect for the interactive version on large screens.

This component was repackaged throughout the Texas A&M Health site and the college sites, both with and without the graphic elements.
